zigbee-table

zigbee
发布日期:   2020-05-31

1. 路由记录表 Route Record Table

Field Name Field Type Valid Range Reference
Network Address Integer 0x0000- 0xfff7 route record的目的网络地址
Relay Count Integer 0x0000 - 0xffff 从集中器到目的节点的中继数量
Path Set of Network Addresses 网络地址集合，表示从集中器到目的节点之间的节点地址集合

2. 邻居表 Neighbor Table

Field Name description
Extended address IEEE地址
Network address 节点网络地址
Device type 节点类型(coordinator router end device)
RxOnWhenIdle idle时是否可以接收 父节点和子节点必须present
End Device Configuration 终端节点的配置
Timeout Counter end device remaining time (sec)
Device timeout end device child timeout(sec)
Relationship 与节点之间的关系
Transmit Failure 指示之前的数据通讯是否成功，数值越大，失败越多
LQI 该节点的链路质量
Outgoing Cost 传出损耗 0表示无效
Age
Incoming beacon timestamp 从邻居接收到的最后一条beacon帧的时间戳
Beacon transmission tiem offset
Keepalive Received 表示自从router重启以来，至少接收到一条来自终端设备的数据？
Extended PAN ID 设备所属网络的PAN ID
Logical channel 设备所在的信道
Depth 邻居节点的tree depth
Beacon order 设备的IEEE 802.15.4信标顺序
Permit joining 设备是否正在允许join
Potential parent 设备是否是潜在的父节点

2.1. 邻居表老化

age字段每过一个nwkLinkStatusPeriod，就会增加，当age字段超过 nwkRouterAgeLimit时就会将Outgoing Cost字段设置为0。当有新的邻居节点时，这个邻居节点就可以被抛弃掉。
end device不会发出link status, 所以这一机制不起作用。

3. 路由表 Routing Table

Field Name size Description
Destination address 2 octets 网络地址或者Group ID
Status 3 bits route的状态
No route cache 1 bit 标志位：目的地址没有存储源路由
Many-to-one 1 bit 标志位：目的地址是发出MTO路由请求的集中器
Route record required 1 bit 标志位：在下一个数据包发送到目的节点前应当先将route record发送到
GroupID flag 1 bit 标志位：目的地址是一个Group ID
Next-hop address 2 octets 下一跳的网络地址

3.2. 路由表中end device的Destination address

根据地址分配方式的不同而不同，当nwkAddrAlloc等于0时表示使用分布式地址分配方式，等于2时表示使用随机地址分配方式。
当使用随机地址分配方式时，路由表中end device的地址为该设备的实际地址。当使用分布式地址分配方式时，路由表中end device的地址为其父节点的地址。

3.3. Status

路由表条目的status如下所示：

num status 状态
0x0 active 有效的
0x1 discovery_underway 正在进行路由发现
0x2 discovery_failed 路由发现失败
0x3 inactive 无效的
0x4 validation_underway 正在核验有效性

4. 路由发现表 Route Discovery Table

Field Name size(octets) Description
Route request ID 1
Source address route request发起方的网络地址
Sender address 这条路由请求的最佳上家，之后的route reply会发给这个节点
Forward cost 从发起方到当前节点的累计路径损耗
Residual cost 从当前节点到目的节点的累计路径损耗
Expiration time 倒数计时器，指示路由发现截止时间
